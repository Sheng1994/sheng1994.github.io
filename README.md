# Sheng Lu Personal Academic Website V5

This version uses only a white reading mode. Dark mode and the reading-mode toggle have been removed.

## Home page sections

The home page now only contains:

1. Awards & Honors
2. Academic Activities
3. Contact

Other home-page sections such as education, experience, skills, research overview, and publication summary were removed from the section area as requested.

Research, Publications, and Blog still remain as independent pages.
