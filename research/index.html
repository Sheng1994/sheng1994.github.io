Research

Research Topics

RFET-Based FPGA Architecture

This research studies how emerging reconfigurable field-effect transistor technologies can be used to build new FPGA architectures. The goal is to connect device-level characteristics with system-level architecture benefits.

Specific Work

  • Designed FPGA architectures using emerging reconfigurable digital technologies.
  • Built technology/system co-optimization workflows for RFET-based FPGA design.
  • Evaluated how RFET logic characteristics affect LUT organization, mapping efficiency, area, and delay.

EDA Algorithms for Reconfigurable Logic

This research focuses on algorithmic support for FPGA implementation, especially when the target hardware contains non-traditional or multi-output logic structures.

Specific Work

  • Developed multi-output LUT mapping algorithms to reduce FPGA logic area.
  • Designed delay-aware packing algorithms for RFET-based FPGA architectures.
  • Investigated dual-output LUT technology mapping and logic decomposition methods.
  • Integrated algorithm design with architecture-level evaluation to quantify practical benefits.

Mixed-Signal Neuromorphic Circuit Design

This research studies energy-efficient neural computation through mixed-signal circuit design, SPICE-level simulation, and reconfigurable neuromorphic architectures.

Specific Work

  • Developed an efficient mixed-signal SPICE simulation platform for on-chip STDP learning.
  • Designed reconfigurable neuromorphic architectures with mapping and placement algorithms.
  • Evaluated energy-efficient analog and mixed-signal neural accelerator design methods.

Semiconductor Testing Automation

This work comes from industry experience in failure analysis and lab automation.

Specific Work

  • Automated semiconductor parameter testing using Python-controlled laboratory instruments.
  • Correlated circuit behavior with process variation and physical defects for root-cause analysis.
  • Improved lab workflows using reusable automation scripts and instrument-control procedures.
