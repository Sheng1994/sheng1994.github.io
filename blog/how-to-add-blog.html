How to Add a New Blog Post

Website guide

  1. Copy research-note-template.html.
  2. Rename it, for example my-new-paper-note.html.
  3. Edit the title, date, and article body.
  4. Add a new link in blog/index.html.
  5. Upload the updated files to GitHub Pages.

Example Entry

<li>
  <h3><a href="my-new-paper-note.html">My New Paper Note</a></h3>
  <p>Short description of the blog post.</p>
</li>
